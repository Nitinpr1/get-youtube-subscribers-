Get Youtube Subscribers API

This is a simple API application build using Node.js, Express.js and MongoDB for database management.

Usages :

  1. To get all the subscribers try "/subscribers".
  2. To get all the subscriber's name and subscribed channel try "/subscribers/names".
  3. To get a specific subscriber by Id try /subscribers/id", for example id="6618cb767a61200b9271a0c0".
  4. For Swagger Documentation try "/api-docs".
